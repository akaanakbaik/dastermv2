# Code of Conduct

## Our Pledge

Kami ingin Dasterm menjadi project yang ramah, aman, dan bermanfaat untuk semua orang.

Semua contributor, user, maintainer, dan peserta diskusi diharapkan menjaga sikap baik, saling menghormati, dan membantu dengan cara yang membangun.

---

## Expected Behavior

Perilaku yang diharapkan:

```text
Bersikap sopan
Menghargai pemula
Memberi kritik yang jelas dan membangun
Membantu menjelaskan masalah teknis
Menghindari hinaan personal
Menghargai perbedaan pengalaman
Tidak menyebarkan data sensitif
Tidak memaksa orang menjalankan command berisiko
```

---

## Unacceptable Behavior

Perilaku yang tidak diterima:

```text
Pelecehan
Hinaan
Spam
Ancaman
Doa buruk atau ujaran kebencian
Menyebarkan token/password/private key orang lain
Mendorong user menjalankan command merusak
Mengirim malware
Menyalahgunakan issue/PR untuk promosi tidak relevan
```

---

## Technical Safety

Karena Dasterm adalah project terminal/Linux, diskusi teknis harus memperhatikan keamanan.

Jangan menyarankan command berbahaya tanpa penjelasan risiko.

Contoh command berisiko:

```text
rm -rf
dd
mkfs
iptables flush
ufw reset
chmod/chown massal
reboot/shutdown tanpa alasan jelas
curl | bash dari sumber tidak jelas
```

Jika command berisiko memang perlu dibahas, jelaskan konteks dan risiko dengan jelas.

---

## Privacy

Jangan meminta user membagikan:

```text
Password
Token
SSH private key
Cookie
KTP/identitas pribadi
File rahasia
Database dump sensitif
.env berisi secret
```

Jika user tidak sengaja membagikan secret, sarankan untuk segera rotate/revoke secret tersebut.

---

## Reporting

Jika melihat perilaku buruk, laporkan melalui GitHub issue atau kontak maintainer.

Saat melapor, sertakan:

```text
Link diskusi/issue/PR
Ringkasan kejadian
Screenshot jika perlu
Dampak yang terjadi
```

Jangan menyebarkan masalah secara berlebihan di tempat lain sebelum maintainer sempat meninjau.

---

## Enforcement

Maintainer dapat:

```text
Memberi peringatan
Menghapus komentar
Menutup issue/PR
Memblokir user dari project
Melaporkan abuse ke GitHub jika perlu
```

Tujuannya bukan untuk menghukum, tetapi menjaga project tetap aman dan nyaman.

---

## Scope

Code of Conduct berlaku di:

```text
GitHub issues
GitHub pull requests
GitHub discussions
Dokumentasi project
Komunikasi terkait Dasterm
```

---

## Final Note

Dasterm dibuat untuk membantu orang memahami server Linux dengan lebih mudah.

Mari jaga project ini tetap ramah, aman, dan berguna.
